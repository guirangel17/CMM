Instructions
************

After extracting the files, in order to compile the code just type:

> make

By typing:

> movGen -h

you can display a list of the options of the generator. Please refer to our ACM MC2R and REALMAN'06 papers for the details of the implementation and configuration of the model. The paper can be downloaded from the project webpage.

As far as the license of the software is concerned, please refer to this webpage:

http://www.cs.ucl.ac.uk/staff/m.musolesi/mobilitymodels

We would be very grateful, if you can send us a copy of or a pointer to any work (papers,  research projects, etc.) where our mobility model has been used.
